компиляция программы:
g++ -std=c++11 -pthread main.cpp -o main

запуск программы
./main n m (где n - количество клиентов, m - количество обработчиков)

Для удобства изначально сгенерировано 10 файлов по 5 функций (то есть первый параметр может быть не больше 10).
Такие числа были выбраны, чтобы можно было за реальное время проверить работу программы. В общем случае, они могут быть любыми.
В папке functions ожидаются файлы, которые будут считываться клиентами, названия этих файлов должны соответствовать шаблону "poly_1.txt ... poly_n.txt" (как в приведенной тестовой выборке).

В программе был реализован алгоритм с использованием блокирующей очереди.
